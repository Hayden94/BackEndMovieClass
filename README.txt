In order to run this program and view my favorites movies the steps include:

	1) Unzip the movies.zip file
	2) Double click on entertainment_center.py to run the program 
		*Another way to run the program is to edit the same file in IDLE(Python GUI) and click Run and then Run Module(or hit the F5 key)
	3) Python will automatically open the html page in your default web browser and create the file within the same folder appropriately named fresh_tomatoes.html
	4) Enjoy!
